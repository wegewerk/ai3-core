=============
Configuration
=============

.. toctree::
    :maxdepth: 1
    :titlesonly:

    EnvironmentVariables
    Logging
    JavaScriptModules

Ai3 Core has no extension configuration form in the TYPO3 backend.
All runtime settings are controlled through environment variables and
TYPO3 configuration files that ship with the package.

See the sub-pages for details on each configuration area.
